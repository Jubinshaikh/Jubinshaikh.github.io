Web Application Programming and Hacking Course Information

This course introduced me to both the fundamentals of web development and the real-world security challenges that come with building modern web applications. I explored key front-end technologies like HTML5, CSS, and JavaScript, along with back-end components such as PHP and MySQL. Beyond development, the course emphasized web vulnerabilities and taught me how to identify and defend against attacks through hands-on labs and hackathons. We also covered essential principles of secure coding, which helped me understand how to write more resilient and secure applications

Overview of Lab 0

Lab 0 was all about setting up a strong technical foundation. In Part 1, I installed and configured an Ubuntu virtual machine, setting up critical tools like net-tools, Git, Sublime Text, Apache2, Wireshark, Pandoc, and even Google Chrome. It was my first step into the WAPH lab environment, and it gave me the confidence to work in a Linux-based web dev setup

In Part 2, I cloned the course’s GitHub repository and generated my own SSH key pair. This allowed me to securely connect to GitHub without needing to enter credentials repeatedly a small but powerful productivity boost I also cloned my private repository and ensured Git actions like pull, push, and commit worked seamlessly from the terminal.

Overview of Lab 2

I started by creating a basic HTML page with my name, headshot, and a simple form. Then I added some JavaScript to make the page interactive for example, showing the current date/time when clicking a button, hiding/showing my email, and displaying both digital and analog clocks

In the next part, I worked with AJAX and jQuery to send and receive data from a PHP backend without refreshing the page. I also experimented with different types of CSS—inline, internal, and external to improve the style and layout of the page.

One of the most fun parts was using public APIs. I connected the page to the JokeAPI to show a random programming joke when the page loads, and used the Agify API to predict someone’s age based on their name. These made the page feel more real and dynamic
